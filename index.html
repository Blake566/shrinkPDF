How It Works

Getting started

  • Open ShrinkPDF.
  • Add one or more PDF files.
  • Choose an output folder.
  • Select a compression quality.
  • Click Compress PDFs.
Common Issues

Files or folders cannot be accessed

On macOS, ShrinkPDF needs access to files you select and the output folder you choose. If saving fails:

  • Select the PDF files again from inside the app.
  • Choose the output folder again.
  • Make sure the destination has enough free space.
Compression Notes

Why some PDFs may not shrink much

Some PDFs already contain optimized text and vector content. In those cases, ShrinkPDF may preserve the original file or reduce size only slightly. Image-heavy scanned PDFs usually compress more.
